# 冷光 COLD LIGHT

> 一首完全由代码实时合成的后朋克轻音乐。
> A 小调 · 132 BPM · 60 小节 · 约 1 分 49 秒 · Web Audio API · 无采样、无外部资源。

---

## 一、这是什么

打开 `index.html`,按下播放,浏览器会从零开始"演奏"这首曲子:
振荡器生成贝斯、吉他与合成器,噪声塑形出鼓组,卷积混响与延迟撑开空间。
页面上方是向 Joy Division《Unknown Pleasures》封面致敬的脉冲可视化——
每一道山脊都是正在发声的频谱,颜色随乐曲段落的情绪变化。

```
音乐(乐理数据) ──► 音序器 ──► 合成器(振荡器/噪声) ──► 效果链 ──► 你的耳机
                     │
                     └────► 可视化(频谱山脉 / 鼓点脉冲 / 段落色彩)
```

## 二、创作心路

### 1. 起点:一个画面

动笔之前先有的不是旋律,是一个场景——

> 凌晨三点,雨刚停。空无一人的街道,路灯把湿漉漉的柏油路照出一层冷光。
> 你走在路中央,脚步声是唯一的节拍器。城市巨大、明亮、且完全与你无关。

后朋克是最适合这个画面的语言:它冷、克制、机械,
但在机械的重复底下,总有一条奔涌的暗流。

### 2. 第一个决定:让贝斯当主唱

传统乐队里贝斯躲在后面,后朋克把它推到台前(Peter Hook 是始作俑者)。
这首曲子没有人声,于是我把"叙事者"的角色整个交给贝斯:

- 它用**不间断的八分音符**行走——那个"匀速走过空城"的脚步;
- 它在每个乐句结尾**爬上三度、五度**做旋律填充——像走路时忽然抬头看了一眼天;
- 副歌里它开始**根音与高八度五音来回跳**——心跳变快,但步速不变。

**克制,是整首曲子的核心情感策略:情绪只准从缝隙里漏出来。**

### 3. 和声:永远不给"回家"的满足感

全曲锁死在 A 自然小调(Aeolian),刻意回避和声小调的导音(G#):

- 没有 G#,就没有强力的属七解决,音乐永远"悬着"——这正是后朋克的疏离感;
- 主歌 `Am → Am → F → G`:i–VI–VII,一次次走到 VII 级的"门口"(G),却从不推门进去(不解决回 Am 的强终止),而是滑回 Am 重新开始——**循环本身成为一种宿命感**;
- 副歌反转为 `F → G → Am`:同样三个和弦,换一个顺序,阴天忽然透出一道光,但光的尽头还是 Am;
- 桥段跌进 `Dm`(iv 级):小调里最阴郁的下属和弦,整首曲子情绪的最低点、水面之下。

### 4. 节奏:机器与人的拉扯

- 主歌是 **Motorik 律动**(借自 Krautrock,后朋克的血缘之一):鼓像传送带一样直线运转,军鼓死死钉在 2、4 拍上;
- 副歌切成**四踩底鼓**——不是为了让人跳舞,是为了让人"不得不往前走";
- 吉他刺用 **3-3-2 切分**(每小节 8 个八分音符分成 3+3+2)。8 是偶数、是对称、是秩序;3+3+2 是在秩序内部制造的不安。**这就是那条暗流。**
- 桥段突然半速:整首曲子唯一一次"允许呼吸"。

### 5. 音色的心理学

| 声部 | 合成方式 | 想要的心理效果 |
| --- | --- | --- |
| 贝斯 | 锯齿波+方波次八度,低通急速关闭 | 有体温的机器,叙事者 |
| 吉他 | 三个失谐锯齿波(合唱感)+点八分延迟 | 每个音都拖着回声——空旷街道的墙面反射 |
| 主音合成器 | 锯齿+方波,慢速颤音 | 远处的人声替身,想说话但只发出音高 |
| 铺底 Pad | 失谐锯齿,慢起音,重混响 | 城市的环境光,一直在但你不注意 |
| 鼓组 | 正弦扫频(底鼓)/噪声塑形(军鼓、镲) | 干燥、致密、不浪漫的骨架 |

### 6. 写作时否决掉的东西

- **否决了大调副歌**:试过把副歌解决到 C 大调,瞬间变成青春片配乐,删;
- **否决了加人声旋律采样**:一旦有"人"出现,孤独就不成立了;
- **否决了华丽的鼓 Fill**:只保留军鼓推子和一次坠落式通鼓——后朋克的鼓手不炫技,他们只负责让传送带不停。

## 三、情感弧线 · 段落导聆

整首曲子是一次"深夜穿过城市"的完整心理过程:

| 段落 | 小节 | 和声 | 织体 | 情绪意图 | 可视化颜色 |
| --- | --- | --- | --- | --- | --- |
| 前奏 INTRO | 1–8 | Am–F–G | 贝斯八分根音,鼓由简入繁 | 出门。路灯次第亮起,世界只剩脚步声 | 冷灰 |
| 主歌 I | 9–16 | i–i–VI–VII | 吉他分解和弦加入 | 匀速行走,观察者视角,情绪封存 | 暗红 |
| 副歌 | 17–24 | VI–VII–i | 四踩+和弦刺+主音线 | 情绪第一次抬头——不是快乐,是"确认孤独"时的释放 | 亮红 |
| 主歌 II | 25–32 | i–i–VI–VII | 同主歌 I + 铺底 | 同一条街第二次经过,多了记忆的重量 | 暗红 |
| 副歌 II | 33–40 | VI–VII–i | 尾部坠落式通鼓 | 把同一句话更用力地再说一遍 | 亮红 |
| 桥段 BRIDGE | 41–48 | iv(Dm)为轴 | 半速,织体抽空,只剩回声 | 沉到水面以下,自我对话,全曲最冷处 | 寒蓝 |
| 终章副歌 | 49–56 | VI–VII–i | 完整编制回归 | 从水底浮上来吸到的那一口气 | 橙红 |
| 尾奏 OUTRO | 57–60 | Am–F–G–Am | 元素逐层退场 | 灯一盏盏熄灭。没有大团圆,只有平静 | 灰 |

**聆听建议:**

1. 戴耳机,音量比平时略小——这首曲子的情绪都在中低频;
2. 第一遍**只跟着贝斯走**,它是主唱;
3. 第二遍注意副歌吉他的 3-3-2 切分,数"1-2-3,1-2-3,1-2";
4. 用页面上的**声部开关**做拆解聆听:关掉鼓,听空间;只留贝斯+鼓,听骨架;
5. 桥段(寒蓝色段落)请不要跳过——整首曲子是为了那 8 小节的下沉而写的。

## 五、技术实现

### 文件

```
music1/
├── index.html   页面结构:可视化画布 / 走带控制 / 声部混音台 / 乐理面板
├── style.css    后朋克视觉:黑底、猩红、窄体大字距、噪点、CRT 扫描线
├── script.js    乐曲数据 + 合成引擎 + 音序器 + 可视化(全部注释)
└── README.md    本文件
```

### 合成信号链

```
振荡器/噪声 → 滤波器 → 包络
      │
      ▼
声部总线(drums / bass / gtr / synth,可独立静音)
      │                          ┌────────────────┐
      ├─(发送)──► 卷积混响 ───────►│                │
      ├─(发送)──► 点八分反馈延迟 ──►│   混音总线      │
      └────────────────────────►│                │
                                 └──── 压缩 ── 音量 ── 输出
```

- **音序器**:经典 lookahead 调度(25ms 轮询,提前 120ms 把事件写进音频时间线),保证节拍稳定不受 UI 卡顿影响;
- **混响**:程序生成的指数衰减噪声作为卷积脉冲响应;
- **延迟**:附点八分(0.75 拍),后朋克吉他的招牌空间感;
- **暂停**:直接挂起 AudioContext,恢复时从同一采样点继续。

### 可视化

- **脉冲星山脉**:实时频谱经对数映射折叠成左右对称的山峰(低频在中央——底鼓每一次踩下,山脉中心就隆起一次),逐帧堆叠成 52 行历史,带景深透视;
- **鼓画同步**:音序器在调度底鼓/军鼓/开镲的同时向可视化推送带精确时间戳的事件——底鼓=画面震动+前排光晕,军鼓=线条增亮,开镲=整幅白闪;
- **段落情绪配色**:前排亮线颜色随段落切换(见上方导聆表),桥段会整个冷下来;
- **待机呼吸**:未播放时山脉以慢速正弦轻微起伏,像睡着的仪器。
