# 艾泽拉斯 · 箱庭巡礼 —— 操作说明

> 单文件离线游戏：双击 `wow-azeroth-dioramas.html` 即可运行（推荐 Chrome / Edge）。
> 首次进入或重置存档后，会先弹出 **职业选择** 界面。

---

## 一、开局：选择职业

| 职业 | 资源 | 定位 |
|------|------|------|
| ⚔ 战士 | 怒气（攻击/受击积攒，脱战衰减） | 近战肉搏，冲锋切入，硬刚 |
| 🔥 法师 | 法力（自动回复） | 远程法术，读条火球+冰箭风筝，闪现保命 |
| 🏹 猎人 | 集中（自动回复） | 远程射击，宠物狼"影牙"替你抗怪 |

职业影响：技能组、武器外观（剑/法杖/弓）、服装、战斗方式。存档后职业固定，想换职业需重置存档。

---

## 二、基础操作

### 移动与视角
| 按键 | 功能 |
|------|------|
| `W / S` | 前进 / 后退（后退减速） |
| `A / D` | 原地转身；**按住右键时**变为左右平移 |
| `Q / E` | 左右平移 |
| `空格` | 跳跃（部分 Boss 技能可以跳跃闪避！） |
| `鼠标左键拖拽` | 只转镜头，不转身 |
| `鼠标右键拖拽` | 转动角色朝向 |
| `左键+右键同按` | 向前奔跑 |
| `滚轮` | 拉近 / 拉远镜头 |

### 界面与交互
| 按键 | 功能 |
|------|------|
| `F` | 与 NPC 对话 / 拾取任务采集物（靠近后按） |
| `Tab` | 循环选中最近的敌人 |
| `1 / 2 / 3 / 4` | 释放技能（也可点击动作条） |
| `H` | 炉石：读条 10 秒回闪金镇（冷却 5 分钟） |
| `M` | 打开 / 关闭世界地图（传送） |
| `B` | 背包 |
| `C` | 角色面板（装备与属性） |
| `Esc` | 关闭面板 / 取消选中目标 |

## 三、战斗

1. **选中目标**：按 `Tab`（目标脚下出现红圈，左上出现目标框）。
2. **自动攻击**：近战贴近约 3 码自动挥砍；猎人在 7~32 码自动射击。武器会自动"拔出/收回"。
3. **技能**（按 1/2/3/4）：

- **战士**：英勇打击（耗怒 20，爆发单体）· 冲锋（6~28 码突进）· 旋风斩（耗怒 25，360° AoE）· **雷霆一击**（耗怒 20，AoE+减速 4 秒+震屏）
- **法师**：火球术（读条 1.5 秒，移动会打断）· 寒冰箭（瞬发+减速 4 秒）· 烈焰新星（近身 AoE+定身）· **闪现**（向前瞬移 12 码，可解除蛛网定身）
- **猎人**：奥术射击（瞬发高伤）· 多重射击（同时射 3 个目标）· **假死**（清空所有怪物仇恨，保命神技）· **震荡射击**（眩晕目标 2.5 秒）

4. **仇恨机制**：怪物头顶弹红色"!"即被仇恨；跑出它的巢区它会回家并回满血。猎人的宠物攻击会把仇恨拉到自己身上。
5. **受击反馈**：被打时屏幕会震动并泛红光；被蛛网缠住/被眩晕等状态会显示在头像框下方的 Buff 栏。
6. **死亡**：血量归零倒地，点"在墓地复活"回到本区域出生点（满血复活）。
7. **脱战回血**：脱离战斗 6 秒后生命缓慢恢复；也可点击背包里的治疗药水使用。

## 四、任务

- 头顶 **金色 !** = 有任务可接；**灰色 ?** = 进行中；**金色 ?** = 可交任务。
- 靠近 NPC 按 `F` 对话 → 接受任务。右侧任务栏实时显示进度。
- 三类任务：**击杀**（打指定怪）、**收集**（找地上闪光的物件按 `F` 拾取）、**探索**（走到指定地点）。
- 有的任务是跨区域的（比如奥格瑞玛的任务要去十字路口完成），完成后记得回去交任务拿经验和金币。

---

## 五、地图与传送（M 键）

羊皮纸地图上有 11 个金色标记，点击即传送（**传送后有一段高空俯冲运镜，点击鼠标可跳过**）：

- **联盟侧**：闪金镇（新手 1-2 级）→ 暴风城 → 铁炉堡
- **部落侧**：奥格瑞玛 → 十字路口（3-4 级）→ 雷霆崖
- **进阶**：达纳苏斯 / 藏宝海湾（5-6 级）→ 达拉然 → 冰冠冰川（7-8 级）
- **副本**：**黑爪洞穴**（建议 6 级以上或买好装备再进）

绿色脉动的标记 = 你当前所在区域。城市是安全区，没有怪物。
每个区域还有隐藏的**子区域发现点**（如"晨曦广场""燧核熔窟"），走到即得 30 经验。

### 小地图与时间（右上角）
- 圆形小地图：**白色箭头**是你（指向朝向），红点=怪物（大红点=Boss），金点=任务 NPC，亮金点=采集物，蓝点=其他冒险者，绿点=宠物
- 小地图上方显示当前**子区域名称**
- 地图外环有一颗**金色小珠**沿钟面转动——那是时刻：游戏内约 8 分钟为一天，有完整的**日出、黄昏、星夜**循环，入夜后村庄窗火会亮起
- 按 `H` 使用**炉石**随时回闪金镇（读条 10 秒，冷却 5 分钟）

## 六、黑爪洞穴（副本）

线性三个厅室，小怪把守，Boss 战有机制：

1. **蛛后·丝影**：会用蛛网把你定身 2 秒（法师可用**闪现**解除）；半血召唤小蜘蛛——先杀小怪。
2. **石皮巨怪**：脚下出现**红圈**时是大地践踏预警，1.7 秒内跑出圈，否则被震飞；残血狂怒，输出加快。
3. **燧核领主**：释放**烈焰新星**（扩散的火环）——**跳跃可以闪避**；脚下点火时立刻走位。

Boss 必掉稀有/史诗装备 + 大笔金币。击败后约 10 分钟刷新。

---

## 七、装备与经济

- **拾取**：怪物掉金币和物品（自动拾取）。白/绿/蓝/紫 = 品质从低到高。
- **装备**：打开背包 `B`，点击武器/肩甲/胸甲/披风即穿上，**外观实时改变**（武器颜色、稀有武器发光、肩甲变大等）。属性看角色面板 `C`。
- **商人**：4 位 NPC 有"购买/出售"按钮——闪金镇旅店老板娘（药水）、铁炉堡铁匠（武器护甲）、藏宝湾里克斯（杂货）、达拉然大法师（高端货）。
  - 商店打开时，**点击自己背包的物品 = 出售**（按 25% 折价）。灰色杂物（獠牙/狼皮等）就是用来卖钱的。
- **升级**：击杀与交任务得经验，升级回满血并提升生命与攻击。

---

## 八、系统

- **存档**：每 15 秒自动保存 + 切换区域保存 + 关闭页面保存（左下角"✓ 进度已保存"）。刷新页面自动读档。
- **重置**：地图面板（M）右下角"重置存档"→ 确认后重新开始，可重选职业。
- **声音**：右下角喇叭按钮开关。首次点击页面任意处后音频才会激活（浏览器限制）。每个区域有独立的氛围配乐与环境音。
- **聊天框**：左下角显示系统消息（升级/发现/Boss 击杀）与"世界频道"的居民闲聊。

---

## 九、新手推荐路线

1. 选职业 → 在**闪金镇**找旅店老板娘和卫兵托马斯做完 4 个任务（约 3 级）
2. `M` 传送**十字路口**，做军需官卡佳的任务链；顺路交掉奥格瑞玛格罗什的野猪任务（约 4-5 级）
3. **达纳苏斯 / 藏宝海湾**继续任务，攒钱在铁炉堡买"精钢长剑"（约 6 级）
4. 进**黑爪洞穴**推三个 Boss，拿蓝装紫装
5. 最后去**冰冠冰川**完成双任务链，挑战 8 级亡灵——毕业！
