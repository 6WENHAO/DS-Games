# 钢铁雷霆 STEEL THUNDER

一款致敬《战争雷霆》陆战玩法的**网页版 3D 坦克射击游戏**。纯前端实现（Three.js + 原生 ES Modules），无需构建工具。

## 快速开始

由于浏览器安全策略（ES Modules / 贴图加载），需要通过本地 HTTP 服务器运行：

- **方式一（推荐）**：双击 `start.bat`，浏览器会自动打开 `http://localhost:8137`
- **方式二**：在本目录执行 `python -m http.server 8137`，然后访问 `http://localhost:8137`
- **方式三**：`npx serve .` 或任意静态服务器

> 推荐使用 Chrome / Edge。首次加载约 30MB 资源。

## 玩法

- 统治模式：夺取并守住中央 **A 点**，消耗敌方票数；击毁敌车也会扣除对方票数
- 我方 5 车（你 + 4 AI）对阵敌方 5 车，双方均有增援；你有 **3 次重生机会**
- 4 辆可选载具：T-34-85 / M4A2 谢尔曼 / 虎式 H1 / IS-2

### 操作

| 按键 | 功能 |
| --- | --- |
| W / S | 前进 / 倒车 |
| A / D | 车体转向 |
| 鼠标 | 炮塔瞄准（两段式准星：白点=视线，绿圈=火炮实际落点） |
| 左键 | 开炮 |
| Shift / 右键 | 狙击镜（滚轮切换 4x/6x/8x） |
| 滚轮 | 第三人称视距 |
| 1 / 2 | 穿甲弹 / 榴弹 |
| Tab | 计分板 |
| M | 放大小地图 |
| Esc | 暂停菜单 |

### 拟真机制（简化自战争雷霆）

- **穿甲判定**：名义装甲 ÷ 入射角余弦 = 等效装甲；穿深随距离衰减；大角度跳弹
- **模块伤害**：弹药架（可殉爆）、发动机（减速/起火）、履带（瘫痪+原地维修）、火炮（装填变慢）
- **弹道**：抛物线弹道 + 曳光，火控自动测距抬炮
- **侦查**：敌车需被我方视线发现后才在地图/标记中显示

## 项目结构（模块化）

```
index.html            入口页面（HUD/菜单 DOM）
styles/main.css       全部界面样式
lib/                  Three.js r160 及官方插件（本地化）
assets/               CC0 素材（模型/纹理/音效/图标）
src/
  main.js             状态机与流程编排（加载→机库→战斗→结算）
  core/               engine 渲染循环 / assets 资源清单 / input / audio / utils
  world/              terrain 地形+多层PBR混合 / world 地图布景+碰撞 / environment 天空光照 / materials 细节着色
  game/               tank 载具实体 / player 玩家控制 / ai 敌军 / projectiles 弹道 / battle 战局 / effects 粒子特效
  ui/                 hud 战斗界面 / menu 机库
```
