围 攻

Besiege Web — 建造中世纪攻城机械，碾碎城堡与骑士

建造模式

左键 点击零件表面放置当前零件（底部工具栏选择）
右键 点击零件删除零件（核心方块不可删除）
R旋转待放置零件的朝向（铰链 / 木板等有方向的零件）
Ctrl+Z撤销
🔧 扳手工具点击已放置的零件，修改其按键绑定 / 速度 / 反转等参数

视角

右键拖动 / 左键拖动空地旋转视角
中键拖动（建造模式也可用 W A S D）平移视角
滚轮缩放

模拟模式（默认按键，可用扳手修改）

空格开始 / 结束模拟
W S动力轮 前进 / 后退
A D转向铰链 / 转向方块
C加农炮开火
F火焰喷射器
H螺旋桨推进
P活塞伸出
V分离器分离
G抓取器松开
T切换 跟随 / 自由 相机
