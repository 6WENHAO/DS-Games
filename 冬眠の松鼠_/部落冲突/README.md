# 部落村庄 · Clash Village

一个致敬《部落冲突 (Clash of Clans)》的网页版基地建设 + 掠夺战斗游戏。
纯原生 HTML / CSS / JavaScript（无框架、无构建工具），所有美术与音效均为 **CC0 公共领域素材**。

## 运行方式

**方式一（推荐）**：直接双击 `index.html` 用浏览器打开即可（Chrome / Edge）。
全部资源均为本地文件，无需联网。

**方式二**：本地服务器（任选其一）：

```
# Python
python -m http.server 8000
# 然后访问 http://localhost:8000

# Node
npx serve .
```

## 玩法说明

| 系统 | 内容 |
|---|---|
| 资源 | 金币（金矿产出）、圣水（圣水泵产出）、宝石（清障碍/升级奖励），点击采集器收取 |
| 建造 | 商店购买建筑 → 拖动摆放 → ✔ 确认；建筑工人有限，可用宝石加速 |
| 升级 | 点选建筑 → 升级；大本营等级解锁更多建筑与等级上限 |
| 军队 | 兵营训练：野蛮人/弓箭手/哥布林/巨人/炸弹人/法师；驻地决定人口上限 |
| 科技 | 实验室研究部队等级；法术工厂酿造雷电法术 |
| 战斗 | 「进攻！」→ 侦察随机敌村（可花金换图）→ 沿红区外部署部队 → 3 分钟内尽量摧毁 |
| 星级 | 摧毁大本营 ★ / 50% ★ / 100% ★；战利品自动入库，奖杯增减 |
| 防御 | 加农炮 / 箭塔(可旋转瞄准) / 迫击炮(范围伤害有射击死角) / 法师塔 / 哨戒塔 / 城墙 |
| 存档 | 自动保存到 localStorage，离线时资源照常生产、建造照常完成 |

操作：鼠标拖动平移，滚轮/双指缩放，点击选择/收集，按住选中的建筑拖动可搬家，ESC 取消。
调试：`F9` 重置存档重新开始；URL 加 `#demo` 可观看自动战斗演示。

## 项目结构

```
index.html            入口（DOM UI 骨架）
css/style.css         全部界面样式（奇幻风 9-slice 面板）
js/
  core/               引擎层：utils / audio / assets / input
  data/               数值配置：config / buildings / troops
  game/               玩法逻辑：state(存档) / village(网格) / economy / training
    battle/           战斗：generator(敌村生成) / pathfind(A*) / units(部队AI)
                      defenses(防御AI+弹道) / battle(总控)
  render/             渲染：camera(等距投影) / fx(特效) / renderer(主渲染)
  ui/                 界面：hud / shop / panels / army / battleui
  main.js             启动 & 主循环
assets/               全部 CC0 素材（图片/音效/字体），详见 assets/CREDITS.md
```

## 技术要点

- 等距(2:1)投影 + 深度排序绘制，地面瓦片预渲染到离屏画布
- A* 寻路支持"穿墙代价"，部队会智能选择绕路或砸墙
- 防御塔独立索敌/弹道/溅射；箭塔按 8 方向旋转帧瞄准
- 帧动画图集（爆炸/烟雾/烟花/星星）+ 粒子系统
- 数据驱动：所有建筑/兵种数值集中在 `js/data/`，方便调整平衡性
