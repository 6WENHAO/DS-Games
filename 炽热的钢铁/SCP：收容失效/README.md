# SCP：收容失效 — Web 同人重制

基于浏览器技术（Three.js + Web Audio + Canvas）从零实现的《SCP – Containment Breach》同人致敬版。
所有贴图、音效均为程序运行时即时生成，无任何外部素材文件；文本为自创概述性写作。

## 运行方式

方式一（推荐）：

```bash
cd 本目录
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

方式二：直接双击 `index.html`（Chrome / Edge 桌面版）。

需要键鼠，建议 Chrome/Edge 最新版。点击“开始游戏”后自动锁定鼠标。

## 玩法

你是 D-9341。SCP-173 收容间的例行清洁因全站门控失灵演变为收容失效。
穿过 轻收容区(LCZ) → 检查点A → 重收容区(HCZ) → 检查点B(特斯拉门) → 入口区(EZ)，
逐级获取 1→4 级钥匙卡（或用 SCP-914 的“精制”档升级钥匙卡），从 B 大门电梯登上顶层平台撤离。

### 收容物

- **SCP-173**：被注视时无法移动；你眨眼或视线离开的每一瞬间，它都在逼近。听到石头刮擦声就别回头太久。
- **SCP-106**：定期从地板中腐蚀而出、穿墙追击。被触碰会坠入口袋维度——8 扇门只有 1 扇能回到设施。
- **SCP-096**：坐在重收容区某条走廊。看到它的脸 = 触发追杀，它会拆掉沿途所有门。别看。跑也没用，所以别看。
- **SCP-914**：放入物品，旋钮选择 粗制/糙制/1:1/精制/极精制，拉杆精炼。“精制”可升级钥匙卡，“极精制”有惊喜也有惊吓。

### 物品

钥匙卡 L1-L5、防毒面具（毒气走廊必备）、S-NAV 导航仪（右上角小地图）、
急救包、滴眼液（暂停眨眼消耗）、收音机、电池、以及多份可阅读的档案文件。

### 提示

- 检查点大门会在数秒后自动关闭，可以用来甩开身后的东西（对 096 无效，它会拆门）。
- 特斯拉门有充能预警音，掐准节奏冲刺通过。
- 停电时所有人都看不见——包括你，但 173 不需要看见你。
- 地图种子相同则布局相同，可在标题界面输入种子复盘。

## 技术说明

- `js/layout.js` 程序化生成三区网格地图（迷宫算法 + 特殊房间放置 + 钥匙卡进度可解性校验）
- `js/map.js` 将布局构建为合并网格的 Three.js 场景（含口袋维度与地表结局场景）
- `js/npcs.js` 三个 SCP 的 AI（视线判定 / 网格寻路 / 事件驱动）
- `js/audio.js` Web Audio 合成全部音效与环境声，广播用 SpeechSynthesis
- `js/textures.js` Canvas 程序化生成全部贴图与物品图标
- 无构建步骤、无网络依赖（Three.js r147 已本地化于 `lib/`）
