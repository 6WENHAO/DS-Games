# 方块拓荒 BLOCKY WILDS

体素沙盒 × 时间循环太空探险的融合致敬之作。**全部美术资源均为原创 SVG 程序化生成**，
**全部音效与配乐均为原创 WebAudio 程序化合成**（Karplus-Strong 拨弦模拟乡村民谣质感），
不包含任何第三方版权素材。仅依赖 three.js（MIT 许可，已本地打包于 `lib/`）。

## 如何运行

直接双击 `index.html` 即可在浏览器中游玩（推荐 Chrome / Edge）。
无需服务器、无需联网。

## 玩法

### 生存模式 · 时间循环
1:1 致敬时间循环太空探险的前期流程：
1. **篝火旁醒来** —— 与长者对话，学习基础
2. **拓荒者的手艺** —— 砍树 → 合成木板/木棍 → 木镐（挖掘/合成教学）
3. **领取装备** —— 博物馆宝箱：宇航服（喷气背包+氧气）、信号镜、古族翻译器
4. **发射密码** —— 登上天文台，天文学家告知太阳将在 22 分钟后超新星爆发
5. **驾船升空** —— 在太阳系中自由飞行，选择星球着陆
6. **追寻古族信标** —— 三颗星球各藏一块石板（挖掘/建造/喷气背包解谜）：
   - 燧沙星：沙层下的古族穹顶（**向下挖掘**）
   - 碎空星：裂隙下的晶洞（**喷气背包下降与返回**）
   - 风暴星：耸入风暴的古塔（**搭方块攀登 + 歇脚台回气**）
7. **深空之眼** —— 集齐三段坐标后，星图上出现新信号，飞向它见证结局

太阳每 **22 分钟**爆发超新星，世界重置——但你的**知识**（密码、坐标、飞船日志）跨越循环保留。
死亡同样只是回到篝火旁。篝火处可**冥想**跳到循环终点。

### 创造模式
无限方块、双击空格飞行、无计时器，自由建造与检验。

## 技术亮点

- **空间弯曲星球**：世界是标准的方块体网格，由自定义顶点着色器将整个空间
  弯折包裹成球——脚下永远是地面，草面永远朝向星球外侧，远处地形随曲率下沉，
  绕星球一圈无缝衔接（x/z 坐标模环绕 + 着色器 `mod` 折叠）。
- **全 SVG 美术管线**：16×16 像素质感的方块贴图、等距方块物品图标、UI、
  头像、字符、驾驶舱、Logo 全部由代码生成 SVG 再栅格化为纹理图集。
- **程序化音频**：方块声按材质合成；喷气背包/飞船引擎/篝火/风为实时循环声源；
  超新星轰鸣为多层噪声+正弦扫频；配乐为 Karplus-Strong 弦乐模型演奏的原创
  D 大调五声民谣主题（篝火/太空/末日三种编排）。
- **体素球星球**：太空视角下的星球是按半径壳生成的立方体聚合网格，与地表
  弯曲世界风格统一。
